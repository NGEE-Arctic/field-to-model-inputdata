# About

This folder is data prepared for the NGEE Modex 2026 Workshop. This was prepared
using tools in the new `temds` package (currently unpublished, but in development
here: https://github.com/uaf-arctic-eco-modeling/Input_production

This data is dowscaled from CRU-JRA climate, using WorldClim and the delta method.

This is a 1 pixel dataset (aka single grid cell, or site).
